
# Bram Welding - Static Website Package (GitHub + Vercel ready)

Isi paket ini:
- index.html, styles.css, app.js
- assets/images/ (dummy product SVG images)
- firebase-config.example.js (rename + fill to enable product upload + storage)
- README ini
- deploy: upload repo ke GitHub, lalu hubungkan repo ke Vercel (static)
- To enable online product storage:
  1. Buat proyek Firebase (console.firebase.google.com)
  2. Aktifkan Firestore (mode Native) dan Firebase Storage
  3. Masukkan konfigurasi Firebase Anda ke file `firebase-config.js` (rename), lalu deploy.
  4. Pastikan rules Firestore/storage diatur aman (misal sementara untuk testing: allow read; allow write for authenticated or limited rules).
